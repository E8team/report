## Install

### 1. Clone the source code or create new project.

```shell
git clone https://github.com/E8team/report.git
cd report
composer install
```

### 3. Installation frontend
```shell
npm install
```

### 4. Compile the frontend resources
```shell
// 开发环境
npm run dev
or
// 生产环境
npm run prod
```
